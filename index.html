This is a demo of the jquery.codeassist plugin, written by Adam Eivy and hosted on GitHub

Basic (Defaults)

Enter some text and type a % character to bring up the menu.

Dollar Sign

Enter some text and type a $ character to bring up the menu.

Square Brackets

Enter some text and type a [ character to bring up the menu.

HTML Brackets

Enter some text and type a < character to bring up the menu.

HTML Open/Close

This is NOT meant to be an HTML helper (there are other libraries for that)--but by a sort of request:

Enter some text and type a < character to bring up the menu.

Next enter the html value and type </.
